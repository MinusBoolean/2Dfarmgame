# Pixel Farm V1 — 设计规格说明书

> V1 目标：从彩色方块原型升级为可玩的像素风格农场游戏。

## 1. 概述

**V1 范围**：农场场景 + 种地系统 + 季节/天气/昼夜 + 体力/背包/商店 + 音效/特效 + 存档。

**不在 V1 范围**（V2/V3）：矿洞、池塘钓鱼、工坊、觅食、果树、温室、出货箱路径、公告板。

**技术栈**：Phaser 3 + TypeScript + Vite，16x16 像素精灵，星露谷风格。

## 2. 场景架构

```
BootScene          — 资源预加载（真实进度条）
UIScene             — 常驻 HUD（金币、日期、季节、天气、工具栏、背包）并行运行
FarmScene           — 农场区域（Tilemap + 作物系统 + 玩家）
```

**场景切换**：V1 只有农场场景，不涉及传送。但架构预留了多场景扩展能力。

**Camera**：
- 主相机跟随玩家（`cameras.main.startFollow(player)`）
- 像素渲染：`pixelArt: true, roundPixels: true`
- 世界边界：64x48 tiles = 1024x768 像素

## 3. 农场世界设计

**尺寸**：64x48 tiles（1024x768 像素）

**布局**：
```
[北边：装饰树木/围栏边界]
[西北：小屋] [中央：耕地区（初始 20x15，可扩展）] [东北：仓库]
[西南：出货箱] [中央偏南：工坊入口（V2）] [东南：装饰花坛]
[南边：通往矿洞/池塘的路（V2 传送点预留）]
```

**Tilemap 层**：
- Ground 层：草地、泥土、路径
- Object 层：建筑、围栏、装饰物、出货箱
- Overlay 层：树冠、屋顶（遮挡玩家）
- Collision 层：不可通行区域

**装饰物**：
- 树木（松树、橡树）— 不同季节有不同颜色
- 围栏（木栅栏）
- 花丛（季节性花朵）
- 稻草人（保护范围内作物）
- 路灯（夜晚发光）

## 4. 种地系统

### 4.1 核心循环

翻地 → 选择种子 → 播种 → 浇水（每天）→ 等待生长 → 收获 → 出售/出货

### 4.2 作物配置

12 种作物，每种有：
- 4 个生长阶段精灵（种子→小苗→大苗→成熟）
- 4 级品质（普通/银/金/铱）
- 生长天数、售价、种子价格、解锁条件
- 适种季节（1-3 个季节）
- 体力消耗（播种/浇水）

| 作物 | 种子价 | 售价(普通) | 生长天数 | 适种季 | 解锁条件 |
|------|--------|-----------|---------|--------|---------|
| 胡萝卜 | 10 | 20 | 3 | 春 | 初始 |
| 小麦 | 5 | 12 | 4 | 春夏 | 初始 |
| 番茄 | 15 | 35 | 5 | 春夏 | 累计 200 金币 |
| 玉米 | 20 | 45 | 6 | 夏秋 | 累计 500 金币 |
| 西瓜 | 30 | 65 | 8 | 夏 | 累计 1000 金币 |
| 土豆 | 12 | 28 | 4 | 春 | 累计 150 金币 |
| 南瓜 | 25 | 55 | 7 | 秋 | 累计 800 金币 |
| 草莓 | 35 | 75 | 6 | 春 | 累计 1500 金币 |
| 蓝莓 | 40 | 85 | 7 | 夏 | 累计 2000 金币 |
| 辣椒 | 18 | 40 | 5 | 夏秋 | 累计 300 金币 |

| 菠菜 | 8 | 18 | 3 | 春秋 | 累计 100 金币 |

### 4.3 品质系统

4 级品质，影响售价倍率：
- 普通（1x）：基础品质
- 银（1.25x）：连续浇水 3 天以上
- 金（1.5x）：连续浇水 7 天以上
- 铱（2x）：连续浇水 14 天以上 + 使用肥料

品质视觉：
- 普通：正常颜色
- 银：银色边框 + 微闪
- 金：金色边框 + 闪烁
- 铱：紫色边框 + 强闪光 + 粒子效果

### 4.4 生长系统

- 生长时间基于游戏天数（非现实时间）
- 每天需要浇水，否则生长速度减半
- 季节结束时，不适应新季节的作物枯死（灰色纹理）
- 枯死的作物需要翻地清除
- 雨天自动浇水

### 4.5 稻草人

- 制作成本：50 金币 + 10 木材（V1 简化：50 金币直接购买）
- 保护范围：半径 5 格
- 每天有 10% 概率乌鸦偷吃无保护的作物
- 稻草人有动画（挥手）

## 5. 时间系统

### 5.1 季节

- 4 季：春（绿色）、夏（明亮）、秋（橙黄）、冬（白色）
- 每季 14 天
- 季节影响：可种植作物、天气概率、背景色调、装饰物外观
- 季节过渡：最后 2 天渐变

### 5.2 天气

| 天气 | 春 | 夏 | 秋 | 冬 |
|------|-----|-----|-----|-----|
| 晴天 | 60% | 70% | 50% | 40% |
| 雨天 | 30% | 20% | 30% | 0% |
| 雷暴 | 0% | 10% | 0% | 0% |
| 雾天 | 10% | 0% | 20% | 0% |
| 雪天 | 0% | 0% | 0% | 60% |

天气效果：
- 雨天：自动浇水，钓鱼概率提升（V2）
- 雷暴：5% 概率闪电击中无稻草人保护的作物
- 雾天：视野降低（Phaser tint 叠加）
- 雪天：作物不生长，移动速度 -20%
- 天气预报：HUD 显示明天天气

### 5.3 昼夜

- 1 天 = 1 分钟现实时间（可调速 0.5x/1x/2x/4x）
- 前 40 秒：白天（明亮）
- 后 20 秒：夜晚（变暗）
- 日出/黄昏有渐变过渡
- 夜晚：整体变暗（深蓝 tint），小屋窗户/灯笼发光
- 夜晚不影响矿洞（V2）
- 回家睡觉或等待到天亮

### 5.4 体力系统

- 初始体力：100
- 每天早上恢复满
- 操作消耗：
  - 翻地：-2
  - 播种：-1
  - 浇水：-2
  - 收获：-1
- 体力 ≤ 10 时：动作变慢，显示警告
- 体力 = 0 时：无法操作
- 食物恢复（V1 简化：商店卖面包 20 金币恢复 30 体力）

## 6. 背包与经济

### 6.1 背包

- 初始 20 格，可花钱扩展到 40 格（每 4 格 500 金币）
- 物品可堆叠（同种+同品质最多 99 个）
- 背包满时：收获/挖矿/钓鱼操作失败，显示"背包已满"
- 物品类型：种子、作物（含品质）、工具（不占背包格）

### 6.2 商店

- 购买种子：左侧面板，显示种子名、价格、适种季节
- 出售作物：右侧面板，显示作物名、品质、售价
- 价格动态：品质影响售价（1x/1.25x/1.5x/2x）

### 6.3 出货箱

- 农场有固定位置的出货箱
- 玩家把物品放入箱子，一天结束时统一结算金币
- 每日结算时显示"今日收入"汇总动画
- 也可以保留商店直接卖（价格 80%，但即时到账）

## 7. 工具系统

### 7.1 工具类型

| 工具 | 功能 | V1 最高等级 |
|------|------|-----------|
| 锄头 | 翻地 | 3 级 |
| 浇水壶 | 浇水 | 3 级 |

（镐子和钓竿在 V2 引入）

### 7.2 工具升级

- 升级需要：矿石（V1 简化为金币）
- 升级效果：
  - 锄头 2 级：蓄力翻 3 格直线
  - 锄头 3 级：蓄力翻 3x3 范围
  - 浇水壶 2 级：蓄力浇 3 格直线
  - 浇水壶 3 级：蓄力浇 3x3 范围
- 蓄力操作：长按 1 秒，消耗体力 x3

### 7.3 操作反馈

- 翻地：屏幕微震(0.5px, 100ms) + 泥土粒子
- 播种：种子落入泥土动画
- 浇水：水花粒子 + 水声
- 收获：屏幕微震 + 金币粒子 + 品质光效
- 升级：全屏闪光 + 音效

| Day 2 | 种一个胡萝卜 | 胡萝卜种子 x3 |

## 9. 农场评级

评级系统提供长期目标：

| 等级 | 条件 | 解锁 |
|------|------|------|
| 铜级 | 总收获 50 个 | 解锁番茄、土豆种子 |
| 银级 | 总收获 200 个 + 总收入 5000 | 解锁玉米、辣椒、菠菜种子 |
| 金级 | 总收获 500 个 + 总收入 20000 | 解锁西瓜、草莓、蓝莓、南瓜、茄子种子 |

## 10. 美术资产

### 10.1 渲染配置

```typescript
const config: Phaser.Types.Core.GameConfig = {
  pixelArt: true,        // 禁用纹理平滑
  roundPixels: true,     // 像素对齐
  antialias: false,      // 禁用抗锯齿
};
```

### 10.2 资产清单

| 类型 | 尺寸 | 数量 | 说明 |
|------|------|------|------|
| 玩家角色 | 32x32/frame | 16帧 | 4方向 x (idle1+walk3) |
| 作物 | 16x16/stage | 48帧 | 12种 x 4阶段 |
| 农场 Tileset | 16x16/tile | ~60帧 | 草地/泥土/路径/围栏/建筑 |
| 装饰物 | 16x16 | ~20帧 | 树/花/稻草人/路灯/出货箱 |
| UI | 可变 | ~20帧 | 按钮/面板/图标/tooltip框 |
| 粒子 | 8x8 | ~10帧 | 金币/水花/泥土/雪花/光效 |
| **合计** | | **~174帧** | |

### 10.3 调色板

统一 32 色调色板（星露谷暖色调）：
- 绿色系（草地、树叶）：#4a8c3f, #6db856, #8fd672
- 棕色系（泥土、木材）：#8b6914, #a67c2e, #c49a48
- 蓝色系（水面、天空）: #4a6fb5, #6b8fd6, #8bb5f5
- 暖色系（作物、果实）: #e8a030, #f0c060, #f8e090
- 灰色系（石头、夜晚）: #404040, #606060, #808080

### 10.4 素材生成流程

1. 使用 AI 生成像素精灵，prompt 包含：调色板、风格描述、尺寸要求
2. 生成后裁剪对齐到网格
3. 检查色调一致性
4. 导入 Phaser（spritesheet 格式）

## 11. UI 设计

### 11.1 HUD（顶部）

- 左上：金币图标 + 数字
- 中上：日期（Day X）、季节图标、天气图标、明天天气图标
- 右上：体力条（绿色→红色渐变）
- 保存指示器："保存中..." 文字，1 秒淡出

### 11.2 工具栏（底部）

- 像素风格面板
- 6 个格子：锄头、浇水壶、食物、空、空、空
- 数字键 1-6 快捷切换
- 选中工具高亮边框

### 11.3 背包面板

- 按 B 键打开
- 像素风格网格布局（4 列 x 5-10 行）
- 物品图标 + 数量数字
- 点击物品：查看 tooltip（名称、描述、品质、售价）
- 右侧：扩展背包按钮

### 11.4 商店面板

- 按 E 键打开（靠近商店建筑时）
- 左侧：购买种子列表（名称、价格、适种季节）
- 右侧：出售作物列表（名称、品质、售价）
- 底部：金币余额

### 11.5 种子选择器

- 播种时弹出（背包中有种子时）
- 显示可用种子列表（图标 + 数量 + 适种季节）
- 点击选择，ESC 取消

### 11.6 设置菜单

- ESC 键打开
- 音乐音量滑块（0-100%）
- 音效音量滑块（0-100%）
- 时间流速选择（0.5x/1x/2x/4x）
- 智能工具切换开关
- 重置存档按钮（需二次确认）

### 11.7 每日结算

- 晚上自动触发（或回家后触发）
- 全屏面板显示：今日收获数、今日收入、今日浇水次数
- 金币飞入动画
- "新的一天" 标题淡入

### 11.8 Tooltip

- 悬停 0.5 秒后显示
- 像素风格边框
- 内容：物品名称、品质图标、描述、售价
- 在背包、商店、出货箱中都生效

## 13. 存档系统

### 13.1 SaveData 结构

```typescript
interface SaveData {
  version: number;           // 存档版本号
  gold: number;              // 金币
  totalEarned: number;       // 累计收入
  totalHarvested: number;    // 累计收获数
  currentSeason: number;     // 当前季节 (0-3)
  currentDay: number;        // 当前天数 (1-14)
  currentWeather: string;    // 当前天气
  tomorrowWeather: string;   // 明天天气
  energy: number;            // 当前体力
  maxEnergy: number;         // 最大体力
  inventorySize: number;     // 背包容量
  inventory: InventoryItem[]; // 背包物品
  farmGrid: TileSaveData[][]; // 农场地块状态
  toolLevels: {              // 工具等级
    hoe: number;
    wateringCan: number;
  };
  unlockedCrops: string[];   // 已解锁作物 ID
  farmRating: string;        // 农场评级 (copper/silver/gold/iridium)
  tutorialDay: number;       // 教程进度
  shippingBin: InventoryItem[]; // 出货箱待结算物品
  settings: {                // 用户设置
    musicVolume: number;
    sfxVolume: number;
    timeSpeed: number;
    smartToolSwitch: boolean;
  };
}
```

### 13.2 存档规则

- localStorage key：`pixel_farm_v1_save`
- 每次操作后自动保存（翻地、播种、收获、买卖等）
- 保存时显示"保存中..."指示器
- 加载时：缺失字段用默认值填充（向前兼容）
- 版本号变化时：执行迁移函数

## 16. 与 V2/V3 的衔接点

V1 架构预留以下扩展点：
- 多场景系统（FarmScene 已隔离，可新增 MineScene/PondScene）
- SaveData 可扩展（缺失字段自动填充默认值）
- CropConfig 可扩展（新增作物只需加配置）
- Tilemap 层结构支持多区域地图
- AudioSystem 支持多场景 BGM
- 工具系统可新增镐子/钓竿
- 背包可新增矿石/鱼类物品类型
