# Pixel Farm V2 — 矿洞与工坊设计规格

> V2 目标：在 V1 基础上加入矿洞探索、工坊制作、公告板任务系统。

## 1. 概述

**V2 范围**：矿洞（5 层独立地图）+ 镐子工具 + 工坊制作 + 公告板任务。

**技术栈**：沿用 V1（Phaser 3 + TypeScript + Vite），新增 MineScene + WorkshopScene + BulletinBoard。

## 2. 矿洞系统

### 2.1 矿洞结构

- 5 层独立地图，每层 32x32 tiles（512x512 像素）
- 入口在农场南边，走到矿洞入口按 E 进入第 1 层
- 每层有楼梯通往下一层（清除楼梯周围的石头后激活）
- 矿洞内有照明系统（玩家周围发光，越深越暗）

### 2.2 矿层随机生成

每层进入时随机生成：
- 60% 石头障碍物（需要镐子清除）
- 15% 矿脉（可挖掘获得矿石）
- 10% 空地（可通行）
- 5% 楼梯（通往下一层）
- 10% 收集品（稀有物品）

### 2.3 矿石种类

| 矿石 | 层级 | 出现概率 | 售价 | 用途 |
|------|------|---------|------|------|
| 铜矿 | 1-2 层 | 40% | 15 | 铜级工具升级 |
| 铁矿 | 2-3 层 | 30% | 35 | 铁级工具升级 |
| 金矿 | 3-5 层 | 20% | 80 | 金级工具升级 |

### 2.4 收集品

每层有 5% 概率出现稀有收集品：
- 化石（售价 200）
- 水晶（售价 300）
- 古代遗物（售价 500）

### 2.5 挖矿机制

- 走到矿石/石头旁边按空格
- 镐子敲击，矿石显示破碎动画（3 次敲击破碎）
- 破碎后掉落物品，自动拾取放入背包
- 高级矿石需要更高级镐子（铜镐挖铁矿，铁镐挖金矿）

### 2.6 矿洞危险

- 石头障碍物：需要镐子清除
- 体力消耗：每次敲击 -3 体力
- 无怪物（V2 简化）

## 3. 镐子工具

### 3.1 镐子等级

| 等级 | 名称 | 敲击效率 | 可挖矿石 | 升级材料 |
|------|------|---------|---------|---------|
| 1 | 木镐 | 1 次/秒 | 铜矿 | 初始 |
| 2 | 铜镐 | 1.5 次/秒 | 铜矿、铁矿 | 铜矿x5 + 500金 |
| 3 | 铁镐 | 2 次/秒 | 铜矿、铁矿、金矿 | 铁矿x3 + 1000金 |

### 3.2 操作方式

- 在矿洞场景中，按 4 选择镐子
- 按空格敲击面前的矿石/石头
- 敲击 3 次后矿石破碎，掉落物品

## 4. 工坊系统

### 4.1 交互方式

- 农场中央偏南有工坊建筑
- 玩家走到工坊旁边按 E 打开制作界面

### 4.2 制作配方

| 物品 | 材料 | 金币 | 说明 |
|------|------|------|------|
| 铜镐 | 铜矿x5 | 500 | 升级镐子到 2 级 |
| 铁镐 | 铁矿x3 | 1000 | 升级镐子到 3 级 |
| 铜锄头 | 铜矿x3 | 400 | 升级锄头到 2 级 |
| 铁锄头 | 铁矿x2 | 800 | 升级锄头到 3 级 |
| 铜水壶 | 铜矿x3 | 400 | 升级水壶到 2 级 |
| 铁水壶 | 铁矿x2 | 800 | 升级水壶到 3 级 |
| 背包扩展 | 铁矿x2 | 500 | 背包 +4 格（最大 40） |
| 面包 | 小麦x2 | 10 | 恢复 30 体力 |
| 果汁 | 胡萝卜x3 | 15 | 恢复 50 体力 |
| 普通肥料 | 铜矿x1 | 20 | 品质 +1 级 |
| 速效肥料 | 铁矿x1 | 50 | 生长时间 -25% |
| 稻草人 | 铜矿x2 | 50 | 保护半径 5 格 |

### 4.3 制作流程

1. 走到工坊按 E 打开界面
2. 左侧显示可制作物品列表（灰色 = 材料不足）
3. 点击物品查看所需材料
4. 点击"制作"按钮，消耗材料和金币
5. 制作成功显示动画 + 音效

## 5. 公告板任务系统

### 5.1 交互方式

- 农场入口附近有公告板建筑
- 玩家走到公告板按 E 打开任务界面

### 5.2 任务生成

- 每天刷新 3 个可选任务
- 任务类型：收获作物、挖矿、钓鱼（V3）、制作物品
- 任务有难度等级（简单/中等/困难）

### 5.3 任务示例

| 任务 | 目标 | 奖励 | 难度 |
|------|------|------|------|
| 收获胡萝卜 | 收获 5 个胡萝卜 | 50 金币 + 评级积分 | 简单 |
| 挖掘铜矿 | 挖掘 10 个铜矿 | 100 金币 + 评级积分 | 简单 |
| 制作肥料 | 制作 3 个普通肥料 | 80 金币 + 评级积分 | 中等 |
| 收获金品质作物 | 收获 3 个金品质作物 | 200 金币 + 评级积分 | 困难 |
| 深入矿洞 | 到达矿洞第 3 层 | 150 金币 + 评级积分 | 中等 |

### 5.4 任务流程

1. 走到公告板按 E 查看可用任务
2. 点击任务查看详情，点击"接取"
3. 任务显示在 HUD 右侧（进度条）
4. 完成目标后自动提示"任务完成"
5. 回到公告板领取奖励

## 6. 场景架构

```
BootScene          — 资源预加载（新增矿洞资源）
UIScene             — 常驻 HUD（新增任务追踪）
FarmScene           — 农场（新增矿洞入口、工坊、公告板）
MineScene           — 矿洞（新场景，5 层）
```

## 7. 存档扩展

```typescript
interface SaveData {
  // ... V1 字段
  mineData: {
    currentFloor: number;        // 当前矿层
    floors: MineFloorData[];     // 每层数据
  };
  workshopRecipes: string[];     // 已解锁配方
  quests: QuestData[];           // 当前任务
  completedQuests: number;       // 已完成任务数
  collectedItems: string[];      // 已收集的收集品
}

interface MineFloorData {
  tiles: MineTileData[][];       // 32x32 矿层数据
  discovered: boolean;           // 是否已探索
}

interface MineTileData {
  type: 'empty' | 'rock' | 'ore' | 'stairs' | 'collectible';
  oreType?: 'copper' | 'iron' | 'gold';
  collectibleType?: string;
  hitsRemaining: number;         // 剩余敲击次数
}

interface QuestData {
  id: string;
  title: string;
  description: string;
  target: { type: string; id?: string; count: number };
  reward: { gold: number; ratingPoints: number };
  progress: number;
  accepted: boolean;
  completed: boolean;
}
```

## 8. 与 V1 的衔接

- 保留 V1 所有功能和存档兼容
- 新增字段自动填充默认值
- 矿洞入口在农场南边，不影响现有布局
- 工坊和公告板作为新建筑加入农场
