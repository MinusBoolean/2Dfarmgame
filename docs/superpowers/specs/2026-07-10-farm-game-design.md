# 2D 农场游戏 — 设计文档

> 日期：2026-07-10
> 版本：v1.0

---

## 1. 项目概述

一款基于 Phaser.js 的 2D 像素风格农场模拟经营网页游戏。核心玩法为种地循环：翻地 → 播种 → 浇水 → 收获 → 出售，通过金币解锁新作物和扩展地块。

### 参考产品
- **QQ农场**：纯种地循环节奏
- **Hay Day**：加工链和订单系统思路
- **星露谷物语**：像素美术风格和瓦片地图呈现方式

## 3. 架构设计

### 3.1 整体结构

```
Phaser 游戏实例
├── 场景管理
│   ├── BootScene        —— 资源预加载、进度条
│   ├── FarmScene        —— 主游戏场景（地图、角色、交互）
│   └── ShopScene        —— 商店界面（覆盖层/弹出面板）
├── 数据层
│   ├── GameState        —— 全局状态管理（金币、背包、已解锁内容）
│   └── FarmData         —— 地块网格数据（每格状态、种植信息）
├── 实体
│   ├── Player           —— 玩家角色（移动、动画、交互）
│   ├── FarmTile         —— 地块（状态切换、作物精灵显示）
│   └── CropConfig       —— 作物定义（纯数据配置）
├── 系统
│   ├── GrowthSystem     —— 生长计时逻辑
│   ├── EconomySystem    —— 买卖金币流转
│   └── SaveSystem       —— localStorage 存档读写
└── UI
    ├── HUD              —— 顶部信息栏（金币、工具栏）
    ├── Toolbar          —— 底部工具切换（翻地/种子/浇水壶）
    └── ShopPanel        —— 商店购买面板
```

### 3.2 场景流转

```
BootScene (加载资源) → FarmScene (主游戏)
                            ↓ 点击商店按钮
                         ShopPanel (弹出层，不切换场景)
```

## 4. 核心玩法设计

### 4.1 地块状态机

```
      [翻地]         [播种]          [生长计时]        [收获]
荒地 ──────→ 已翻地 ──────→ 幼苗期 ──────────→ 成熟期 ──────→ 荒地
                                                    ↑            
                                              枯萎(未浇水超时)
```

- **荒地**：可交互翻地（消耗体力/工具耐久，v1 不消耗）
- **已翻地**：可播种，需选择背包中的种子
- **幼苗期**：显示种子对应的幼苗精灵，开始生长计时
- **成长期**：每过一段时间切换生长阶段精灵
- **成熟期**：显示成熟作物精灵，可收获
- **枯萎**：v1 暂不实现枯萎，后续可选加入

### 4.2 作物配置（v1）

| 作物 | 买入价 | 卖出价 | 生长时间 | 解锁条件 |
|------|--------|--------|----------|----------|
| 萝卜 | 5 | 15 | 30秒 | 初始 |
| 小麦 | 10 | 30 | 60秒 | 初始 |
| 玉米 | 25 | 70 | 90秒 | 累计赚 100 金币 |
| 番茄 | 50 | 130 | 120秒 | 累计赚 300 金币 |
| 西瓜 | 100 | 250 | 180秒 | 累计赚 800 金币 |

### 4.3 经济系统

- **初始金币**：100
- **收入来源**：收获作物后到商店出售
- **支出**：购买种子、扩展地块
- **地块扩展**：初始 4×4（16格），付费逐步解锁至最大 8×8（64格），具体定价在实现时确定

## 5. 数据模型

### 5.1 作物配置（可扩展）

```typescript
interface CropConfig {
  id: string;
  name: string;
  growthStages: number;          // 生长阶段数
  growTime: number;              // 基础生长时间（秒）
  buyPrice: number;
  sellPrice: number;
  unlockGold: number;            // 解锁所需累计收益
  spriteKey: string;             // 精灵图 key
  // === 预留给后续扩展 ===
  mutations?: MutationRule[];    // 间种变异规则
  adjacentBonus?: AdjacentRule;  // 相邻地块加成
  harvestDrop?: DropTable;       // 收获掉落表
}
```

### 5.2 存档结构

```typescript
interface SaveData {
  gold: number;
  totalEarned: number;           // 累计收益（用于解锁判断）
  inventory: Record<string, number>;  // { cropId: count }
  farmGrid: TileData[][];        // 地块网格数据
  unlockedTiles: number;         // 已解锁地块数
  unlockedCrops: string[];       // 已解锁作物 ID 列表
  lastSaveTime: number;          // 最后存档时间戳
}

interface TileData {
  state: 'empty' | 'plowed' | 'growing' | 'mature';
  cropId?: string;
  plantTime?: number;            // 播种时间戳
}
```

## 6. 操作方式

| 操作 | 方式 |
|------|------|
| 角色移动 | WASD / 方向键 |
| 地块交互 | 鼠标点击前方地块 |
| 切换工具 | 底部工具栏点击 或 数字键 1-5 |
| 打开商店 | 点击商店按钮 或 按 E 键 |

---

## 7. 视觉风格

- 俯视 2D 像素风格
- 瓦片地图组成农场场景（草地、土地、围栏、房屋）
- 角色有四个方向的行走动画（或静态精灵 + 朝向）
- 作物有种子 → 幼苗 → 成长 → 成熟四个生长阶段精灵

---

## 8. 数据持久化

- 使用浏览器 localStorage 存储 `SaveData`
- 每次地块状态变更、金币变动时自动保存
- 游戏启动时自动加载存档，无存档则初始化为新游戏
- 提供"新游戏"按钮清除存档

---

## 9. 后续扩展规划（不在 v1 范围）

以下功能在设计阶段已预留扩展接口，将在后续版本中实现：

- **间种变异**：相邻地块种特定作物组合时，概率产出特殊果实
- **季节系统**：春夏秋冬影响作物生长速度和可用作物
- **动物养殖**：鸡、牛、羊，产出鸡蛋、牛奶、羊毛
- **加工系统**：磨坊、奶酪机，将原材料加工为高价值产品
- **订单系统**：NPC 订单，按要求交付作物获得额外奖励
- **钓鱼系统**：水边钓鱼小游戏
- **NPC 社交**：村民送礼、好感度、剧情
- **天气系统**：雨天自动浇水
- **背包系统**：升级背包容量
- **成就系统**：里程碑奖励
